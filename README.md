## `> ls ./projects`

- 🧠 **MemGuard** `Final Year Project · 2025–Present`
  
  Hybrid logic-layer integrity monitor that detects **LPCI (Logic-Layer Prompt Control Injection)** attacks in agentic AI persistent memory (ChromaDB / Qdrant). Stage 1 uses regex + semantic hashing (all-MiniLM-L6-v2 / ONNX) for synchronous screening. Stage 2 runs Llama-3-8B-Instruct Q4_K_M via llama.cpp for async deep auditing. Mapped to MITRE ATT&CK and OWASP LLM Top 10.
